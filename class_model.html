Member Data Documentation

vector<Cell> Model::listOfCells
private

Vector sequence containing all the cell objects in the model.

Note
Each object in the object list is stored at the index of its ID - This was done for speed in locating the object needed. However this does have consequences - every ID in the file must be consecutive, or else the object will attempted to be added to a point outside the range of the std::vector. In the future, if this is no longer acceptable, then store ID as a parameter of the object itself. This would mean that each object could be stored at any point in the list, but would take longer to search for the object with a particular ID.
